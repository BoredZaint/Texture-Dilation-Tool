A browser-based tool for dilating textures with transparency.

## Features
- Edge-based smooth dilation algorithm
- Infinite dilation mode
- Semi-transparent pixel handling  
- Live magnifying glass preview
- Customizable dilation amount and blend smoothness
- Client-side processing (no uploads, completely private)
- Preserves original image quality

## Usage
Simply drag and drop a PNG image with transparency, adjust the settings, and process. Download the result with automatic filename generation.
